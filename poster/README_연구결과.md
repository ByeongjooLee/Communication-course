# 📢 온라인 갈등 및 혐오 확산 메커니즘 분석 프로젝트

## 커뮤니케이션 수업 기말과제 연구 모음

현대 디지털 미디어 환경에서 발생하는 사회적 갈등과 혐오 표현의 생성·확산 과정을 실증적으로 분석한 프로젝트입니다.

---

## 📖 프로젝트 개요

본 프로젝트는 뉴스 기사, 유튜브 댓글, 온라인 커뮤니티(디시인사이드 등) 데이터를 수집하여 텍스트 마이닝과 프레임 분석을 수행하였습니다. 이를 통해 **불확실한 정보가 어떻게 '사실'로 둔갑하고 집단적 혐오로 고착화되는지** 규명하고자 했습니다.

### 핵심 연구 질문

> "온라인 공간에서 갈등과 혐오는 어떻게 만들어지고 확산되는가?"

---

## 📂 연구 주제 및 주요 사례

본 프로젝트는 **사회적 갈등 이슈**와 **인물 관련 루머 이슈**로 나누어 총 8가지 사례를 심층 분석했습니다.

### 1️⃣ 사회·정치적 갈등 이슈

| 사례 | 분석 내용 |
|------|----------|
| **의사 파업 논쟁** | '구조·원인(의사)' 프레임과 '윤리·정당성(정부)' 프레임의 충돌 과정 분석, 댓글 공간의 갈등 증폭 메커니즘 파악 |
| **철도 노조 파업** | '적자 속 성과급'이라는 언론 프레임이 대중의 부정적 여론을 형성하고, 구조적 원인에 대한 논의를 차단하는 과정 분석 |
| **MZ 세대 갈등** | 'MZ'라는 용어가 단순한 세대 지칭에서 2023년을 기점으로 비난과 혐오를 동반한 '세대 갈등 프레임'으로 변질된 시점과 양상 추적 |
| **이태원 참사 담론** | 불완전한 단서가 정치적 프레임과 결합하여 집단적 확신과 혐오로 이어지는 '6단계 연쇄 과정' 규명 |

### 2️⃣ 인물·연예계 루머 및 혐오 이슈

| 사례 | 분석 내용 |
|------|----------|
| **연예인 갑질 논란** | 뉴스의 갈등 프레임이 유튜브의 자극적 서사를 거쳐 커뮤니티의 혐오로 이어지는 3단계 확산 구조 확인 |
| **훈련사 갑질 의혹** | 해명 영상 이전 단계에서 루머가 가장 빠르게 확산되며, 렉카 유튜버들이 '인성 문제'로 논란을 확장하는 메커니즘 분석 |
| **마약 의혹과 '과거 끼워맞추기'** | 대중이 과거의 인터뷰나 표정을 현재의 의혹에 끼워 맞춰 "원래 그런 사람"이라며 혐오를 정당화하는 심리 기제 분석 |
| **혐오의 합리적 포장** | 댓글의 다수가 '중립적 어조'를 띠면서도 실제로는 혐오 표현을 내포, 혐오가 '합리적 판단'처럼 확산되는 현상 포착 |

## 📡 플랫폼 간 확산 구조

혐오는 개인의 일탈이 아니라 플랫폼 간 상호작용을 통해 **구조적으로 생산**됩니다.

```
┌──────────────┐      ┌──────────────┐      ┌──────────────┐
│     뉴스     │ ──▶ │    유튜브    │ ──▶ │   커뮤니티   │
│   (점화)     │      │   (증폭)     │      │   (고착)     │
├──────────────┤      ├──────────────┤      ├──────────────┤
│ • 갈등 프레임 │      │ • 자극적 서사 │      │ • 집단 동조  │
│ • 따옴표 보도 │      │ • 렉카 콘텐츠 │      │ • 혐오 표현  │
│ • 헤드라인   │      │ • 알고리즘   │      │ • 낙인 고착  │
└──────────────┘      └──────────────┘      └──────────────┘
```

## 🛠 분석 방법론 (Methodology)

본 연구들은 **데이터 기반의 정량적 분석**과 **6단계 연쇄 모델을 활용한 정성적 분석**을 병행하였습니다.

### 📊 데이터 수집

| 플랫폼 | 수집 방법 | 도구 |
|-------|----------|------|
| **뉴스** | 언론 보도 수집 | 빅카인즈(BigKinds) |
| **유튜브** | 영상 정보 및 댓글 수집 | YouTube Data API |
| **커뮤니티** | 게시글 및 댓글 크롤링 | 디시인사이드, 에펨코리아 크롤러 |

### 🔬 분석 기법

| 기법 | 세부 내용 |
|-----|----------|
| **텍스트 마이닝** | 형태소 분석, 명사 추출, 빈도 분석 |
| **네트워크 분석** | 단어 임베딩(Word2Vec), 연관어 분석, 의미 연결망 시각화 |
| **감성 분석** | KoBERT 모델을 활용한 긍정/부정/중립 감성 분류 및 시계열 추적 |
| **프레임 분석** | 뉴스 보도의 프레이밍 방식 비교 분석 |

### 📐 분석 프레임워크

**6단계 연쇄 분석 모델**을 적용하여 혐오 확산 과정을 체계적으로 분석:

1. **단서 발생**: 불완전한 정보의 출현
2. **의혹 생성**: 추측과 해석의 시작
3. **서사화**: 스토리로 재구성 (주로 유튜브)
4. **프레임 결합**: 정치적/사회적 맥락과 결합
5. **집단 확신**: "다 아는 사실"로 고착화
6. **혐오/낙인 고착**: 낙인과 배제의 정당화

## 💡 핵심 연구 결과 (Key Findings)

### 1. 혐오의 구조적 확산

> 혐오는 개인의 일탈이 아니라 **뉴스(점화) → 유튜브(증폭) → 커뮤니티(고착)**의 플랫폼 간 상호작용을 통해 구조적으로 생산됩니다.

### 2. 중립적 어조의 역설

> 욕설보다 **'점잖은 척하는' 중립적 어조의 혐오 댓글**이 더 높은 공감을 얻으며 여론을 주도합니다.

### 3. 과거 끼워맞추기 (Past Fitting)

> 불확실한 정보를 사실로 믿기 위해 **과거의 행적을 재해석**하여 현재의 비난을 정당화하는 경향이 뚜렷합니다.

### 4. 프레임의 대립과 고착

> 의사 파업이나 세대 갈등에서 보듯, **서로 다른 질문**("왜 그랬나" vs "그래도 되나")을 던지는 프레임의 충돌은 대화를 단절시키고 갈등을 심화시킵니다.

## 🚀 제언 (Suggestions)

### 1. 미디어 리터러시 강화

> 정보의 진위 여부를 넘어, **프레임이 구성되는 방식**과 **혐오가 확산되는 구조** 자체를 비판적으로 인식해야 합니다.

**실천 방안**:
- 뉴스의 프레이밍 방식 분석 교육
- 온라인 담론의 확산 구조 이해
- 감정적 반응 전 맥락 파악 습관화

### 2. 언론의 윤리적 보도

> 갈등을 조장하는 **'따옴표 보도'와 자극적 헤드라인**을 지양하고, 맥락 중심의 보도가 필요합니다.

**개선 방향**:
- 양비론적 프레임 지양
- 구조적 원인에 대한 심층 보도
- 선정적 헤드라인 자제

### 3. 플랫폼의 책임

> 혐오 표현이 수익으로 연결되는 **알고리즘 구조를 개선**하고, 자극적 콘텐츠 노출을 조절하는 장치가 요구됩니다.

**필요한 조치**:
- 자극적 콘텐츠 추천 알고리즘 개선
- 혐오 표현 모니터링 강화
- 건전한 토론 문화 유도 장치
