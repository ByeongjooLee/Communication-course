# 🔍 혐오와 갈등은 만들어지는가?

## 텍스트 마이닝을 활용한 온라인 담론 분석 교육

온라인 텍스트 데이터를 분석하여 혐오 표현과 갈등 패턴을 탐지하는 교육용 텍스트 마이닝 프로젝트입니다.

## 📚 학습 목표

1. 온라인 텍스트 데이터 수집 및 전처리 방법 이해
2. 감성 분석을 통한 혐오 표현 탐지
3. 토픽 모델링으로 갈등 주제 파악
4. 시각화를 통한 패턴 발견

## ✨ 주요 기능

- **텍스트 전처리**: KoNLPy를 활용한 형태소 분석 및 키워드 추출
- **감성 분석**: 규칙 기반 + 딥러닝(KoBERT) 감성 분류
- **갈등 탐지**: 갈등 유발 가능성 분석
- **토픽 모델링**: LDA를 활용한 주제 분류
- **혐오 패턴 탐지**: 일반화, 비하, 선동, 편가르기 표현 감지
- **시각화**: 워드클라우드, 분포 차트, 대시보드

## 📊 분석 파이프라인

```
데이터 수집 → 텍스트 전처리 → 형태소 분석
                              ↓
        감성 분석 ← 갈등 탐지 ← 혐오 패턴 탐지
                              ↓
        토픽 모델링 → 시각화 → 종합 리포트
```

## 🔬 분석 기법

### 감성 분석
| 방법 | 설명 |
|-----|------|
| 규칙 기반 | 긍정/부정/갈등 단어 사전을 활용한 점수 계산 |
| KoBERT | 딥러닝 기반 한국어 감성 분류 |

### 갈등 유발 가능성
| 수준 | 기준 |
|-----|------|
| High | 갈등 유발 표현 2개 이상 |
| Medium | 갈등 유발 표현 1개 |
| Low | 갈등 유발 표현 없음 |

### 혐오 표현 패턴
| 유형 | 예시 키워드 |
|-----|------------|
| 일반화 | 항상, 걔네, 그쪽, 너희, 다들 |
| 비하 | 한심, 멍청, 무식 |
| 선동 | 각성, 분노, 일어나, 심판 |
| 편가르기 | ~충, ~놈, 좌파, 우파 |

### 토픽 모델링 (LDA)
- 문서-단어 행렬 생성 (CountVectorizer)
- 잠재 디리클레 할당(LDA)으로 주제 추출
- 토픽별 주요 키워드 분석
